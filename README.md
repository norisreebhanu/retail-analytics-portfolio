## Project: Retail Sales & Inventory Performance Analytics

### Objective
Analyze retail sales data to track business KPIs, evaluate SKU performance, and identify actionable insights for decision-making.

### Key Questions Answered
- What are the total revenue and profit trends?
- Which SKUs drive the most revenue and margin?
- Which products underperform and require attention?
- How does performance vary by category or region?

### Tools Used
- SQL (KPI calculations, aggregations)
- Python (data cleaning and analysis)
- GitHub (version control and documentations)

### Status
- Calculated core business KPIs including revenue, profit, orders, and units sold using SQL
- Analyzed top and bottom performing SKUs to identify revenue drivers and loss contributors
- Evaluated category-level performance trends
- Python (pandas, numpy) for exploratory data analysis and KPI validation


## Business Impact

This analysis demonstrates how retail data can be translated into actionable insights to support:
- Revenue optimization
- SKU rationalization
- Inventory and pricing decisions
